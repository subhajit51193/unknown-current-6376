# MonsterIndia.com Clone

Monster is a global online employment solution for people seeking jobs and the employers who need great people.
With an vision in mind we started working on the project to create clone of the website with as much function as possible.

Some of the pages are static due to time frame. More focus was on the design part for this Project.


## Features

- Signin/Signup and Login options
- Dynamic Home page with function and great design
- Job search option with attach option for CV
- Search jobs by location feature
- Information on Career Tips page
- Premium Services and its information
- Chatbot with precoded answers according to qustions.
- Cart page with an option to give coupon code



## Optimizations

As this was a limited time project so all of the pages are not dynamic.
We will continue to work on it....
